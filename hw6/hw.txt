Домашнее задание:
1. Создать базу данных на любую категорию;
2. Создать 2 таблицы, где в каждой будет primary key и в одной из них foreign key, который будет ссылаться на другую таблицу. При этом должно использоваться каскадное удаление данных.
3. Написать запросы на вставку данных (INSERT) для обеих таблиц.
4. Написать запросы на выборку данных (SELECT) из таблиц, а именно: запрос на выборку записей со всеми столбцами в результирующей таблице; только выбранные столбцы; используя фильтр данных (... WHERE ???), запрос с использованием LIKE;
5. Запрос на изменение данных (UPDATE).
6. Написать запрос на удаление данных (DELETE), с использованием фильтра и без.
7. Всю базу данных нужно экспортировать в файл (имя_базы.sql), для этого воспользуйтесь утилитой mysqldump.
